What is Poter?

Poter is a full-featured download manager.

Supports downloading various resources such as HTTP, FTP, BT, magnet links, IPFS, etc. It also supports batch downloading and automatic recognition of the content type being downloaded.

Web3.0 storage returns data sovereignty back to individuals, while the trading platform promotes the valuable transmission of data.

Ecologys

Ultimate Speed,
No Restrictions

Poter focuses on providing users with the ultimate download speed and supports various resources downloads such as HTTP, FTP, BT, etc. The tracker list is automatically updated daily to ensure resource activity. Through IPFS, data communication can be achieved without network restrictions.

Portable: Search can be used, anywhere

Building a blockchain-based search engine on the IPFS protocol stack allows Poter users to search for the content they need more quickly and easily. Through bounties, users can pay for scarce resources.

Permanent Storage and
Free Transmission

Use a unique data processing method to isolate user private data and network data. The separation of public and private storage will maximize the use of physical storage space while ensuring the security of user data, and can also save users a lot of storage costs.

Features

Clean Interface

The Poter downloader adheres to the minimalist page style and is dedicated to functional applications, making user operations more convenient.

BitTorrent and Magnet

The Poter downloader supports various download protocols, including BitTorrent and Magnet.

Web3 Data Storage

Poter combines IPFS to achieve decentralized storage services and stores data persistently. Each user has 5GB of free space, and users can upgrade their space based on their needs.

Blockchain and NFT

Poter combines blockchain technology to decentralize data transactions. Users can trade NFTs, and business partners can discover their own NFTs on the platform and offer them for sale to users.

Istio Service-mesh Integration

Fission integrates with Istio, which is an open platform to connect, manage and secure micro-services. With Istio, users also get the ability to monitor function usage and trace request latency through dashboards.

Wide Language Support

Fission is extensible to any programming language of your choice. Python, NodeJS, Go, C#, PHP are supported today, but you can build your own custom containers if you need to.

Token Economics

Build Serverless API Backends

Create API backends for web and mobile apps without managing servers. Just write functions, and map them to HTTP routes. Fission takes care of the rest: deployment, routing, scalability, availability. Use Kubernetes’ service discovery and networking to interoperate with other services, like Redis, Postgres, Etcd etc.
